This project was supposed to create a functional, self-moving trash bin. Instead, it created a bin that has a mind of its own (and that mind is currently very confused).
###  Built With
* [Webots](https://cyberbotics.com/) - The open-source robot simulator.
* [Python](https://www.python.org/) - The controller logic.

###  Repository Structure
* `worlds/` - The `.wbt` file where the bin currently sits, refusing to move.

### How to Run (Or Watch It Not Run)
1.  Install **Webots**.
2.  Clone this repository.
3.  Open `worlds/trash_bin_world.wbt`.
4.  Press the **Play** button.
5.  Observe the lack of movement and contemplate the laws of physics.
